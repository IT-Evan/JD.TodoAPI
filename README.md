JD.TodoAPI

介绍

ASP.NET Core教程示例 - 任务管理项目.

软件架构

ASP.NET Core / Razor Pages / EF Core / SQLLocalDB

安装教程

1. 打开项目:
Visual Studio打开JD.TodoApi.sln.
2. 运行项目:
F5运行即可.

使用说明

项目功能菜单:
任务管理, 增/删/改/查任务信息.
